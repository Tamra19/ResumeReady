# ResumeReady
ResumeReady is an AI-powered web application made using django, HTML, CSS and bootstrap designed to help users create professional and effective resumes. By analyzing job descriptions and user-provided information, it suggests relevant keywords, skills, and resume formats to increase a candidate's chances of landing an interview.

## Features
- User authentication
- Resume Building
- Resume Analyzing
- Friendly UI
- SQLite supported

## Tech Stack
- **Backend:** Python, Django
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** SQLite (Default django database)
- **Tools:** Git, VS Code, google-gemini-2.5-flash

## Installation
- Clone the repository
- Create and activate virtual environment
- Install dependencies from requirements.txt
- Apply migrations
- Run the server

## Usage 
- Open your browser and go to http://127.0.0.1:8000/
- Create a new account or login as admin
- Start exploring the features (register, login, make resume, analyze resume, etc.)
